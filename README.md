# 스터디 방식

1. 매주 스터디 시간 전까지 정해진 분량의 책을 읽어옵니다. 책을 읽으면서 각 쪽에서 가장 중요하다고 생각되는 부분에 줄을 긋습니다. 
2. 스터디 시간에 각 쪽마다 줄을 그은 부분을 공유합니다. 
3. 추가로 논의하고 싶은 부분에 대해서 논의합니다.
